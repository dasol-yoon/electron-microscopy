# Ptychography

For the experiment, record the global meta data:

- keV 300 keV
- convergence angle
- spot size

For region of interest, record the following

- Defocus (nm)
- Mag (Mx)
- Scan steps
- dwell time (μm)
- Camera length
- Data collection notes
- Mono
- Current on flucam: 75 pA
- Note (interesting shape? zone axis?)
